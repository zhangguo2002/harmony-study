# 1.1 环境搭建

## 第一步：下载 DevEco Studio

1. 打开官网下载页：https://developer.huawei.com/consumer/cn/deveco-studio/
2. 选择 **Windows 版本**，下载最新的 DevEco Studio（5.x，对应 HarmonyOS 5）
3. 文件较大（约 1GB+），耐心等待

---

## 第二步：安装 DevEco Studio

1. 双击安装包，按向导安装
2. 安装路径**不要有中文或空格**，例如：`D:\DevEco-Studio`
3. 其余选项默认即可

---

## 第三步：首次启动 & 配置 SDK

启动后进入初始配置向导：

1. **设置 SDK 安装目录**（避免中文路径）
2. 勾选下载 `HarmonyOS SDK`（必选）
3. 等待 SDK 下载完成（如速度慢，可在 `Settings > HTTP Proxy` 配置代理）

---

## 第四步：创建第一个项目

菜单 `File > New > Create Project`，选择模板 `Application > Empty Ability`：

| 字段 | 填写示例 |
|------|---------|
| Project name | `MyFirstApp` |
| Bundle name | `com.example.myfirstapp` |
| Save location | `D:\harmonyos-projects\MyFirstApp` |
| Compile SDK | `5.0.0` |
| Model | `Stage`（推荐） |
| Language | `ArkTS` |

点击 `Finish`，等待工程初始化（首次需下载依赖，约需几分钟）。

---

## 第五步：运行项目

### 方式 A：模拟器（推荐新手）

1. `Tools > Device Manager`，点 `+` 新建手机模拟器（Phone，API 12）
2. 启动模拟器
3. 点击工具栏 ▶ 按钮运行

### 方式 B：真机

1. 手机开启开发者模式（设置 > 关于手机，连续点击版本号 7 次）
2. 开启 **USB 调试**，连接电脑授权
3. 选中设备，点击 ▶ 运行

---

## 项目目录结构

```
MyFirstApp/
├── AppScope/
│   ├── app.json5              # 应用名称、版本、包名等全局配置
│   └── resources/             # 应用图标等全局资源
├── entry/                     # 默认模块（HAP）
│   └── src/main/
│       ├── ets/
│       │   ├── entryability/
│       │   │   └── EntryAbility.ets    # 应用入口 Ability
│       │   └── pages/
│       │       └── Index.ets           # 首页
│       ├── resources/                  # 字符串、图片等资源
│       └── module.json5                # 模块配置
└── build-profile.json5                 # 工程构建配置
```

## 常见问题

| 问题 | 解决方法 |
|------|---------|
| SDK 下载很慢 | 配置代理 / 换网络 |
| 模拟器启动失败 | BIOS 开启 VT-x 虚拟化，检查是否启用 Hyper-V |
| 项目初始化卡住 | 等待 Gradle 下载完成，或配置国内 Maven 镜像 |
