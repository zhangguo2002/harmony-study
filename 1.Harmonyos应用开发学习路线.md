# HarmonyOS 应用开发学习路线

## 阶段一：基础入门

### 1.1 环境搭建
- 下载并安装 DevEco Studio（HarmonyOS 官方 IDE）
- 配置 HarmonyOS SDK
- 创建第一个 Hello World 项目
- 了解模拟器与真机调试

### 1.2 ArkTS 语言基础
- TypeScript 基础语法回顾
- ArkTS 扩展语法特性
- 装饰器（Decorator）机制
- 状态管理基础：`@State`、`@Prop`、`@Link`

### 1.3 ArkUI 声明式 UI
- 基础组件：`Text`、`Button`、`Image`、`TextInput`
- 布局组件：`Column`、`Row`、`Stack`、`Flex`、`Grid`、`List`
- 组件样式与属性链式调用
- 自定义组件（`@Component`）

---

## 阶段二：进阶开发

### 2.1 页面路由与导航
- `router` 页面跳转与参数传递
- `Navigation` 组件导航
- 页面生命周期管理

### 2.2 状态管理进阶
- `@Observed` 与 `@ObjectLink`
- `@Provide` 与 `@Consume` 跨组件共享
- `AppStorage` 应用级状态
- `LocalStorage` 页面级状态

### 2.3 网络与数据
- HTTP 请求（`@ohos.net.http`）
- 数据持久化：首选项（Preferences）、关系型数据库（RDB）
- JSON 数据解析

### 2.4 常用系统能力
- 媒体（图片、音视频）
- 通知与提醒
- 定位服务
- 权限申请与管理

---

## 阶段三：高级应用

### 3.1 UIAbility 与 Stage 模型
- Stage 模型架构理解
- UIAbility 生命周期
- Want 与 AbilityStage

### 3.2 多设备适配
- 一次开发，多端部署概念
- 自适应布局与响应式布局
- 不同设备形态适配（手机、平板、智慧屏）

### 3.3 动画与交互
- 属性动画、转场动画
- 手势识别
- 自定义绘制（Canvas）

### 3.4 性能优化
- 懒加载（`LazyForEach`）
- 组件复用
- 渲染性能分析

---

## 阶段四：工程实践

### 4.1 项目架构设计
- 模块化开发（HAP / HAR / HSP）
- MVVM 架构模式实践

### 4.2 测试与发布
- 单元测试与 UI 测试
- 应用签名与打包
- 上架华为应用市场（AppGallery）
